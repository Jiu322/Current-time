# 实时时间显示页面

这是一个极简的实时时间显示页面，自动每秒更新当前时间。

## 功能特点

- 自动实时更新时间（每秒刷新）
- 响应式布局适配不同设备
- 简洁现代的视觉设计
- 使用原生 JavaScript 实现，无需依赖

## 页面结构

- **时间显示**：在页面中央显示格式为 `HH:MM:SS` 的当前时间
- **自适应字体**：字体大小自动适配不同屏幕尺寸
- **暗色系设计**：黑色文字搭配白色背景，清晰易读

## 技术实现

- 使用 HTML5 结构
- CSS Flex 布局实现居中显示
- JavaScript 实现时间获取和自动更新
- 使用 `padStart` 方法确保时间格式统一
- 使用 `setTimeout` 和 `setInterval` 实现精确的秒级更新

## 使用方法

1. 直接打开 `index.html` 文件
2. 页面会自动显示并实时更新当前时间

这个项目非常适合用作简单的网页时钟实现，或者作为学习 HTML/CSS/JavaScript 基础的示例项目。
